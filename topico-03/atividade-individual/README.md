# Atividade Prática Individual - Tópico 3

## Nível realizado
Nível 2 - Intermédio

## Objetivo
Criar, publicar e validar um pequeno serviço web em ambiente Linux, utilizando um servidor web e documentando todas as etapas realizadas.

## Ambiente utilizado
- Sistema Operativo: Ubuntu 24.04
- Ambiente: VirtualBox
- Servidor Web: Nginx

## Rota de publicação
Nginx

## Ficheiros criados
- index.html
- sobre.html
- style.css
- comandos.txt
- publicacao.md
- validacao.md
- README.md

## URLs testados
- http://localhost/topico-03/
- http://localhost/topico-03/sobre.html

## Evidências produzidas
- Estrutura de diretórios criada
- Instalação e validação do Nginx
- Publicação do site
- Página inicial publicada
- Página Sobre publicada
- Testes de acesso realizados

## Dificudades encontradas
- Erro inicial na cópia dos ficheiros para o diretório de publicação.
- Necessidade de instalar e configurar o Nginx.
- Correção da codificação UTF-8 para apresentação correta dos caracteres acentuados.
- Organização da estrutura de diretórios da atividade.
