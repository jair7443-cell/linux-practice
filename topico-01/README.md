## Diferença entre VM, VPS e Infraestrutura em Nuvem

### VM (Máquina Virtual)

Sistema virtual executado dentro de um computador físico através de software como VirtualBox.

### VPS (Virtual Private Server)

Servidor virtual alojado num datacenter e acessível pela Internet.

### Infraestrutura em Nuvem

Conjunto de recursos computacionais disponibilizados sob demanda por fornecedores cloud, permitindo escalabilidade e elevada disponibilidade.
