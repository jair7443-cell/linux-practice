# Linux Practice

Repositório criado para armazenar os exercícios práticos do módulo **Linux e  Segurança na Cloud**.

## Objetivo

Documentar a evolução das atividades práticas realizadas durante a formação, incluindo administração Linux, segurança de sistemas, monitorização, backup, recuperação e continuidade operacional.
