acesso remoto por ssh

o que é ssh - SSH ( Secure Shell) é um protocolo que permite aceder remotamente a servidores linux de forma segura através de uma ligação criptografada.

Que informaçoes seria necessária para aceder a um servidor por ssh:
- utilizador: n ome da conta do servidor 
-endereço IP: endereço do servidor 
-porta: normalmente 22
- palavra - passe ou chave SSh

Exemplo de comando de ligação:
-ssh utilizador@endereco_ip
